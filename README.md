# Truth-or-Dare-Game
The Truth or Dare Game is a fun web-based application developed using HTML, CSS, and JavaScript. The application randomly generates truth questions and dare challenges when users click the corresponding buttons.

# Truth or Dare Game

## Project Overview

The Truth or Dare Game is a simple interactive web application that randomly displays truth questions or dare challenges.

The project is built using HTML, CSS, and JavaScript.


# Features

- Random truth questions
- Random dare challenges
- Interactive buttons
- Responsive design

# Technologies Used

- HTML5
- CSS3
- JavaScript

# Run Project

Open index.html in browser.
